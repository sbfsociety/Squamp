# Squamp
A platformer game made in C++ and SFML. More info in the README.md.

Requirements for build:
  gcc version 15.2.0

if you don't care about the build you just need this folder structure:
  Squamp
  |- /cmake-build-debug
  |     |-/assets
  |     |-/Squamp.exe
  |-Graphics.hpp
  |-Audio.hpp
  |-System.hpp
